Introduction
Please read this first

This is a draft version of the Delta Flyer build-manual and is not the final manual.

In this document I'm going to walk you trough building the Flyer in rought steps. Please be aware that you would need some experience building printers to follow these instructions.

First of all, you will need some tools:.

  • Soldering Iron or a heatset-press to insert heatsets into the printed parts.
  • Epoxy and CA glue
  • 2 and 3mm hex screwdrivers
  • PC with microSD card reader

This has been written from memory, so please let me know if I've missed something.

1
Prepare Arms and Printed Parts
Assembling the delta arms and removing supports

Glue the arms together using the M3x30 threaded rods between the joints and the rods on the printed jig. Use 2xM3x20 in the jig to allign everyhing. Keep the joints oriented identically on each end.

‚ö†Ô∏è Important

Be extra carefull when gluing these together, do them one at a time on a single printed jig for the best result.

Prepare all printed parts by removing all the built-in supports from:

  • Top_Frame(middle) (light blue piece)
  • Top_Frame_A and B (gray piece)
  • Top_Frame_C (gray piece)
  • Carriage (gray piece)

‚ö†Ô∏è Important

Ensure all support material is completely removed before assembly to prevent fitment issues.

2
Install Heatsets
Installing heatsets into frame components

Insert headsets into the following pieces:

  • Updated_Top_Frame_A.stl
  • Updated_Top_Frame_B.stl
  • Updated_Top_Frame_C.stl
  • Updated_Top_Frame_Middle.stl
  • 2 x Updated_Top_Frame_side_with_support.stl
  • Updated_Bottom_Frame_A.stl
  • Updated_Bottom_Frame_B.stl
  • Updated_Bottom_Frame_C.stl
  • 2 x Updated_Bottom_Middle.stl
  • Updated_Bottom_Frame_Center.stl
  • Effector.stl
  • TR6_Hotend_Mount_LDO.stl
  • 3 x Tensioner.stl
  • Top_Cover_A.stl
  • Top_Cover_B.stl
  • Top_Cover_C.stl
  • Handle.stl
  • 2 x Hinge.stl

‚ÑπÔ∏è Tip

Press headsets in firmly and ensure they are seated flush with the printed part surface.

3
Assemble Bottom Frame Corners
Installing motors, bearings, and pulleys

Start by assembling each of the 3 bottom corners.

Insert the F695 bearing into the lower frame corner. Put the GT2 pulley on the motor with the setscrew facing towards the motor. Dont tension the setscrew yet. Attach the motor to the frame corner using 4x M3x30 screws.

‚ö†Ô∏è Important

Ensure the pulley setscrew is positioned towards the motor for proper belt alignment.

4
Assemble Top Frame Corners
Installing idlers and endstops

On the top frame corners, attach the GT2 idler with a M5x16 screw.

Then attach the optical endstop with the black protrusion facing toward the M5 screw head using 2x M3x12 screws.

‚ÑπÔ∏è Note

The endstop orientation is critical for proper homing functionality.

5
Attach Extrusions and Rails
Mounting 2020 extrusions and MGN9 rails

Attach the 2020 extrusion to the top frame corner using 4x M5x10 screws and T-nuts.

Then attach the MGN9 rail using 5x M3x8 screws and T-nuts.

Finally, loosely attach the bottom corner piece to the extrusion using 4x M5x10 screws and T-nuts.

‚ö†Ô∏è Important

Make sure the carriages dont slide off the rail when moving it around. Keep the rubber stoppers in for as long as possible.

6
Install Panel Covers
Attaching corner panel pieces

Attach the 4 corner panel pieces to each upright using 2x M5x10 on each panel, but do not attach the two front corner panels on the "B" corner - these are attached later after the door hinges.

üí° Tip

Now you can make sure all the upright assemblies are identical in length between the top and bottom frame pieces.

‚ÑπÔ∏è Note

The front lower panel cover on the "A" upright has a hole for a magnet.

7
Install Carriages and Belt System
Mounting carriages and routing belts

Attach the carriages to the MGN9 rails using 3x M3x6 screws.

Using the Apex clips, fasten the belt to the tensioner. Put the tensioner into the carriage (apex clip down) and route the belt down around the pulley, back up through the carriage, around the top idler and back into the carriage, attached with another apex clip.

Use a M3x30 screw to tension the belt to the carriage.

üí° Tip

Don't over-tension the belts initially - you can fine-tune tension after the printer is assembled.

8
Assemble Frame and Install Bed
Bolting frame together and mounting heated bed

Bolt the frame together. Remember to have the middle frame pieces without headsets on the outside towards the front.

Add the small sticky feet to the bottom of the frame.

Attach the heated bed to the frame using 3x M3x16 screws and 3x M4 thumbscrews as spacers between the bed and the frame.

‚ö†Ô∏è Note

The bed may arrive pre-assembled - verify the assembly state before proceeding.

9
Install Motor Covers and Electronics Panels
Adding protective covers and mounting panels

Attach the 3 motor covers using M3x6 screws. Note that the one with the notch goes on the back "C" corner.

Attach the top electronics panels - these are marked. Text goes up.

‚ÑπÔ∏è Tip

Check the panel markings carefully to ensure correct orientation.

10
Assemble and Install Hotend
Mounting the hotend assembly to the effector

Attach the TR6_hotend_mount to the Effector using 4x M3x8 screws.

Attach the hotend to the TR6_hotend_mount using 2x M3x16 screws.

Attach the 3010 hotend fan using 3x M3x16 screws.

Route the hotend and 5015 fan wires through the cable relief hole on the left side and attach the 5015 fan using a M3x20 screw.

Bolt the effector to the carriages using the arms and 12x M3x20 screws.

11
Install Extruder and PTFE Tube
Mounting the BMG extruder

Attach the back left panel, then attach the BMG extruder to it using 2x M3x10 screws and heat inserts as nuts on the inside.

üí° Tip

This is a good time to cut and install the PTFE tube to the proper length.

12
Prepare Top Electronics Cover
Assembling cover and installing magnets

Prepare the top electronics cover by bolting it together using 3x M3x8 screws and gluing in the 6 magnets.

Now is also a good time to glue in the magnets on the top frame, keeping polarization in mind.

‚ö†Ô∏è Important

Check magnet polarity before gluing to ensure proper attraction between cover and frame.

13
Install DZ01 Board and RGB Strip
Mounting the control board and lighting

Install the DZ01 board in the electronics bay using 4x M3x6 screws.

Install the RGB strip to the inside of the chamber using 2 zip ties on the top cover (labeled location).

14
Wiring and Klipper Installation
Connecting electronics and installing firmware

Connect all the wires according to the wiring diagram.

Install Klipper on the DZ01 and run through the first setup process using the Klipper Installation Manual.

‚ÑπÔ∏è Reference

Refer to the separate Klipper tutorial for detailed software setup instructions.

15
Install Wire Covers
Adding cable management covers

After initial setup, install the 2 wire covers on the inside of the left panel using 8x M3x6 screws.
